Скрипт просто меняет значения в зашифрованном файле, в процесс игры не инжектируется! 

Файлы settings.ini и CameraDistanceDota2.exe должны находится в одной папке.

Перед первым запуском в settings.ini указывать путь к файлу client.dll по умолчанию такой : C:\Program Files (x86)\Steam\steamapps\common\dota 2 beta\game\dota\bin\win64\client.dll

Запускать через ярлык CameraDistanceDota2.exe.

В случае успешного нахождения и замены будет выведено соответсвующее сообщение.

В файле settings.ini можно выставить нужное значение. 

В случае корректности введенных данных, программа выведет сообщение об успешной замене и закроется через 3 секунды, в противном случае будет выведено сообщение об ошибке и закроется через 3 секунды. 

Только не модифицированный файл можно изменить, если выводится сообщение об успешной замене, а камера не изменилась - проверьте целостность файлов.
